1. slice/splice

- slice(start,end) : start ~end-1 까지의 요소를 추출
- splice(index, n, a, b, c) : index위치에서 n개를 삭제하고 a,b,c요소를 추가


2. concat/join/reverse

- a.concat(b,c) : 여러 개의 배열 요소를 하나로 합친다.
- a.join : 한 개의 배열요소를 하나의 문자열로 합친다.
- a.reverse() : a배열의 순서를 뒤집기



3. sort

arr = [ 234, 21, 150, 11, 55, 501]
- a.sort() : 자릿수 크기별 sort
- 오름차순 11, 21, 55, 150, 234, 501
내림차순 501, 234, 150, 55, 21, 11
11, 150, 21, 234, 501, 55 // 자리수로 sort한다.
- a.join : 한 개의 배열요소를 하나의 문자열로 합친다.
- a.reverse() : a배열의 순서를 뒤집기
