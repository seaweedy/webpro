1. Date객체 생성

- new Date()
- new Date(98765678978) : 1970.1.1 기준으로 경과한 날짜
- new Date("2010.02.23")
- new Date(2010,01,23,12,30,12)
- 월은 0월부터 시작하므로 숫자로 Date객체를 표시할 때 하나 작게 표시



2. Date객체 생성

- now = new Date();
- now.toLocaleString() : 날짜와 시간
- now.toLocaleDateString() : 날짜
- now.toLocaleTimeString() : 시간




3. 요일구하기

- now = new Date();
- day = now.getDay() : 요일
